# 🗣️ Voice Commands Reference

This document details the interaction model for the **My Cinema** Alexa Skill. The skill supports both **English** and **French** natively by detecting the locale of the incoming request.

## 🎙️ Interaction Model Overview

The skill is built around specific **Intents** that trigger the middleware logic. The following table maps user commands to the corresponding intent and required parameters.

| Action | Intent Name | Required Slots | Description |
| :--- | :--- | :--- | :--- |
| **Play Movie** | `PlayMovieIntent` | `MovieName`, `MovieYear` | Search and stream a movie via TMDB Helper. |
| **Play TV Show** | `PlayTVShowIntent` | `ShowName`, `Season`, `Episode` | Search or resume a specific series. |
| **Resume Show** | `ResumeTVShowIntent` | `ShowName` | Fetches the *Next Up* episode from Trakt.tv. |
| **Change Source** | `ChangeSourceIntent` | - | Forces a source selection menu for the active media. |
| **Trigger Patch** | `TriggerPatcherIntent` | - | Manually runs the `Fen Light` auto-patcher logic. |

## ⚙️ Technical Logic & Parameters

### 1. Source Selection Mode
The parameter `SourceMode` (mapped to `SourceMode` slot in the interaction model) allows the user to override the `PLAYER_DEFAULT` configuration and switch to `PLAYER_SELECT` (the `fenlight_select.json` file in Kodi).
* **Logic:** When the user includes terms like *"manually"* or *"avec choix"*, the `force_select` attribute is set to `True` in the Flask session, triggering the addition of `&autoplay=false` to the TMDB Helper URL.

### 2. Trakt.tv Integration
The `ResumeTVShowIntent` performs a multi-step check:
1. **Search:** Retrieves the TMDB ID via `search_tmdb_show()`.
2. **Lookup:** Connects to `api.trakt.tv` using the bearer token stored in `/app/data/trakt_tokens.json`.
3. **Progress:** Fetches the `next_episode` object from the `/shows/{trakt_id}/progress/watched` endpoint.

### 3. Error Handling
If an intent fails (e.g., Kodi is offline, or the content is not found), the skill uses the `get_text()` helper to pull localized responses from `translations.json`:
* **Kodi Offline:** Triggered if `is_kodi_responsive()` returns `False`.
* **Content Error:** Triggered if the JSON-RPC `Player.GetItem` returns an empty result while the user requests a source change.

### 4. Customizing the Interaction Model
You can find the full Interaction Model JSON schema in the `/alexa_speech_assets` folder of the source code. Ensure your Custom Slots (e.g., `MovieName`, `ShowName`) are updated in the **Alexa Developer Console** to reflect your library's media titles for better recognition accuracy.
