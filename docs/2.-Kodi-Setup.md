# Kodi Configuration Guide: TMDB Helper Integration

To enable voice-controlled playback and source selection, **My Cinema** leverages the *TMDB Helper* add-on as an abstraction layer. You must create two custom "Player" JSON files within your Kodi installation. These files tell TMDB Helper how to communicate with the *Fen Light* add-on.

## 1. Locate your Player Directory

Depending on your operating system, navigate to the following directory in your file manager (via SFTP or File Manager+):

*   **Android TV:** `/Android/data/org.xbmc.kodi/files/.kodi/userdata/addon_data/plugin.video.themoviedb.helper/players/`
*   **LibreELEC / OpenELEC:** `/storage/.kodi/userdata/addon_data/plugin.video.themoviedb.helper/players/`

*Note: If the `players` folder does not exist, create it manually.*

---

## 2. Create the Player Files

Create two new text files in the directory mentioned above and name them as follows.

### A. Automatic Playback Player (`fenlight_auto.json`)
This file triggers *Fen Light* using its default settings (no source selection menu).

```json
{
    "name": "Fen Light (Auto)",
    "plugin": "plugin.video.fenlight",
    "action": "play",
    "method": "override"
}
```

### B. Manual Source Selection Player (`fenlight_select.json`)
This file forces *Fen Light* to present a source selection menu, allowing you to choose your stream manually.

```json
{
    "name": "Fen Light (Select)",
    "plugin": "plugin.video.fenlight",
    "action": "play",
    "method": "override",
    "args": "source_select=true"
}
```

---

## 3. Configuration Verification

Once these files are saved, ensure your environment variables in your `docker-compose.yaml` match the filenames you just created:

```yaml
environment:
  - PLAYER_DEFAULT=fenlight_auto.json
  - PLAYER_SELECT=fenlight_select.json
```

### Important Notes:
1.  **Restart Kodi:** After adding the files, a simple restart of Kodi is recommended to allow TMDB Helper to register the new custom players.
2.  **Permissions:** On LibreELEC/OpenELEC, ensure the `kodi` user has read permissions for these files.
3.  **Testing:** You can verify the players are active by attempting to play a movie via the TMDB Helper GUI; you should see both "Fen Light (Auto)" and "Fen Light (Select)" in your player list.

**My Cinema** will now use these definitions to translate your voice commands (e.g., *"Play [Title] manually"*) into specific instructions for the TMDB Helper API.
